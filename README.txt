# FoodCoach: The Proactive Diet Agent

FoodCoach는 AI 기반 능동형 영양 관리 에이전트입니다.

## 주요 기능
1. Vision AI: 사진으로 식단 자동 인식
2. Proactive Agent: 식사 시간 알림 및 맞춤 제안
3. 영양 성적표: 실시간 영양 섭취 분석 및 위치 기반 메뉴 추천
4. 자가 진화 엔진: 사용자 피드백 기반 추천 고도화

## 기술 스택
- Frontend: Next.js
- Backend: Python
- AI: Google Gemini, OpenClaw

## 실행 방법
1. web 디렉토리에서 npm run dev 실행
2. execution 디렉토리에서 python 스크립트 실행
