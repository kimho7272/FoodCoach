---
description: Update README documentation when features change
---

새로운 기능이 추가되거나 기존 기능이 변경될 때마다 README.md 및 README.txt를 최신 상태로 업데이트해야 합니다.

1. **변경 사항 파악**: implementation_plan.md 또는 새로 추가된 파일을 확인하여 변경된 내용을 파악합니다.
2. **README.md 업데이트**: 상세한 기능 설명, 기술 스택, 프로젝트 구조 등을 업데이트합니다.
3. **README.txt 업데이트**: 주요 내용을 요약하여 텍스트 파일로 반영합니다.
4. **Git Commit & Push**: 변경된 리드미 파일을 저장소에 푸시합니다.
